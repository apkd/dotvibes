Amend the last commit.

Quickly check the last commit's contents:
`git show --stat HEAD`

If the current changes seem unrelated to the last commit, create a brand new commit instead, instead of amending unrelated changes on top.

For more specific instructions, read `.../skills/commit/references/commit.md` if you haven't viewed that document before.
Skip otherwise.
